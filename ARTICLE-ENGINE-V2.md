# Savingio Article Engine V2

## LOCK
내 문제 → 가장 궁금한 답 → 계산 → 결과 이해 → 내 상황 → 실제 행동 → 놓치기 쉬운 것 → 공식 기준 → 다음 행동 → FAQ.

## UI LOCK
목차는 세로 목록이 아니라 한 줄 가로 스크롤 내비게이션입니다. PC와 모바일 모두 같은 구조를 사용합니다.

## CONTENT 기준
각 글은 요약 카드 4개, 본문 핵심 섹션 6개, 상황별 사례 4개, 비교표, 체크리스트 7개 이상, FAQ 5개 이상, 다음 행동과 관련 계산기를 포함합니다. 짧은 계산기 소개문 수준의 글은 생성하지 않습니다.

## 운영
`data/article-configs-v2.json`의 콘텐츠만 교체하고 HTML 구조는 변경하지 않습니다. 새 글은 기존 페이지를 복사하지 않고 V2 빌더로 생성합니다.
