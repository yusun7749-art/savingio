# Savingio Brain OS v6

## 고정 원칙
사용자는 카테고리가 아니라 해결하려는 일을 선택한다.

## 최상위 행동 구조
1. 돈 아끼기
2. 받을 돈 찾기
3. 세금 처리하기
4. 급여·일 처리하기
5. 생활 문제 해결
6. 바로 계산하기

## 연결 흐름
문제 발견 → 이해 → 본인 조건 확인 → 계산·조회 → 실행 → 결과 확인 → 후속 행동

## Navigation 규칙
- 현재 페이지가 속한 경로만 자동으로 연다.
- 계산기는 별도 행동 영역에 모으되 원문 링크는 중복 생성하지 않는다.
- 검색은 제목과 대·중·소분류를 함께 탐색한다.
- PC는 고정 Explorer, 1180px 이하는 드로어로 전환한다.
