# Savingio Factory V2.049

## Safe Release Scope

- `factory/config/release_scope.json`에 지정된 파일만 자동 스테이징합니다.
- `git add .`을 사용하지 않습니다.
- 기존에 스테이징된 파일이 있으면 Release를 차단합니다.
- Release Scope 밖의 기존 변경 파일은 건드리지 않습니다.
- 캐시, 로그, 임시 파일은 Release Scope에 넣을 수 없습니다.
- 삭제 파일은 명시적으로 허용하지 않는 한 자동 커밋하지 않습니다.
- 전체 테스트와 Deployment Integrity가 통과해야만 Commit과 Push가 실행됩니다.
