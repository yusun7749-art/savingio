# Savingio Engine DNA Workboard

## 1. Article Engine
- 상태: 행동 흐름 적용 완료
- 다음 성형: 계산형·절차형·비교형·자격형·절약형·문제해결형 분기
- 정보 밀도 기준: 핵심 답변, 조건, 예외, 숫자/근거, 실행 단계, 후속 행동이 모두 존재해야 통과

## 2. Navigation Engine
- 가로 목차 유지
- 좌측 Explorer는 대분류 → 중분류 → 소분류 → 글/계산기 구조
- 기본 화면에서는 현재 글의 경로만 자동으로 펼침
- 다른 대분류는 사용자가 눌렀을 때만 펼침
- PC 238px, 1180px 이하에서는 모바일 드로어 전환

## 3. Site Flow Engine
문제 발견 → 개념 이해 → 본인 조건 확인 → 계산/조회 → 결과 해석 → 실행 → 후속 점검 → 연관 주제 탐험

## 4. Curiosity Engine
관련 글 연결은 제목 유사도가 아니라 다음 질문을 기준으로 구성한다.
- 먼저 읽을 글
- 함께 확인할 글
- 계산할 도구
- 결과 다음 행동
- 놓치기 쉬운 예외
- 정기적으로 다시 볼 글

## 5. QA Engine
- 제목의 답이 초반에 있는가
- 조건과 예외가 분리되어 있는가
- 일반론으로 분량만 채우지 않았는가
- 본문을 다른 글에 그대로 붙일 수 있는 문장이 남아 있지 않은가
- FAQ가 본문 반복이 아닌가
- 계산기·관련 글 링크가 정확한가
- 읽은 뒤 사용자가 다음 행동을 알 수 있는가

## 6. Site Explorer
- 현재 Sprint: Navigation DNA와 UI 성형
- 완료 조건: 전체 페이지 공통 적용, 현재 위치 강조, 본문 비침범, 모바일 정상, 링크 QA
