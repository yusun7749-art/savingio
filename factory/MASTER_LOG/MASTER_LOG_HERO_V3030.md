# V3.030 Savingio 메인 Hero 최초 시안 복원 MASTER LOG

최종 갱신: 2026-07-19 KST

## 작업 배경

사용자는 Savingio 메인 Hero의 기존 선·점·네트워크형 배경이 과하고 이질적이라고 판단했다. 여러 기하학 레퍼런스를 비교한 끝에, 기하학은 주인공이 아니라 배경을 구성하는 재료여야 한다는 방향을 확정했다.

핵심 사용자 표현:

- `기하학은 배경이지 주가 아니야.`
- `들어가자마자 편안하잖아. 이질감 없고.`
- `벽지 바꿔달랬더니 인테리어를 다 뜯어버렸네.`
- `기하학이 아니라 기이해.`
- `기준이 되는 이미지를 정확하게 분석하고 똑같이 못하면 최대한 비슷하게 구현해줘.`

## 최종 기준 이미지

사용자가 가장 마음에 들어 한 최초 메인 시안을 공식 기준 이미지로 확정했다.

기준 시안 특징:

- 전체 아이보리·크림 배경
- 샴페인 골드, 웜베이지, 매우 옅은 쿨그레이 조합
- 화면 밖으로 잘리는 대형 원형과 반원
- 좌우 대각선·사선·흰색 선·얇은 골드선
- 일부 작은 도트 패턴
- 중앙은 넓은 여백을 유지
- 큰 명조 계열 2줄 타이틀
- `가장 빠르게!` 골드 강조
- 긴 원형 검색창과 네이비 검색 버튼
- 계산기·정부지원·보험·절약팁·생활정보·블로그 6개 카드
- 각 카드 안 둥근 아이콘 배경과 실제 SVG 선형 아이콘
- 하단 신뢰·속도·응원 3칸 안내 바
- 카드와 안내 바는 불투명 흰색이 아니라 배경이 은근히 비치는 반투명 글래스 구조

## 진행 과정과 실패 기록

### 1차 실패 — 배경만 바꾸라는 요청에서 범위 이탈

- 배경만 수정해야 했으나 새 홈페이지 시안을 만들거나 전체 구조를 바꾸려 했다.
- 사용자는 이를 `벽지를 바꿔달랬더니 집을 때려 부쉈다`고 표현했다.
- 이후 범위 이탈 방지 기준으로 LOCK했다.

### 2차 실패 — 승인된 배경과 구조를 제대로 구현하지 못함

- Hero를 가운데 정렬하고 배경 이미지를 넣었지만 최초 시안의 전체 구성과 달랐다.
- 기존 아래 콘텐츠가 일부 사라지거나 축소돼 사용자가 `아래 있던 애들이 다 집 나갔다`고 지적했다.
- Hero 높이 문제가 아니라 실제 아래 섹션이 삭제된 것이 핵심이었다.

### 3차 실패 — 최초 시안 구조만 흉내 냄

- 명조 제목, 검색창, 6개 카드, 하단 안내 영역을 넣었지만 다음이 부족했다.
- 기하학 배경이 거의 보이지 않았다.
- 카드가 불투명 흰색이라 배경이 이어지지 않았다.
- 카드 아이콘 자리에 작은 문자·기호가 들어가 원이 비어 보였다.
- 원 안 아이콘의 크기와 정렬이 맞지 않았다.
- 사용자는 `니 눈에 같은 거로 보여?`라고 지적했다.
