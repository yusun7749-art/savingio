# Savingio 메인 상황검색 연결 수정

## KST 시각
- 2026-07-23

## 사용자 지적
- 메인 `지금 겪는 상황을 선택하세요` 카드가 `집`, `자동차`, `돈`처럼 지나치게 넓은 검색어를 사용해 관련 없는 글까지 노출됐다.
- 예: 집 카드에서 관리비·종합소득세·노인 일자리 등이 함께 노출되고, 사고·보험 카드에서 에어컨 연비 절약·과태료가 섞였다.
- 검색창은 `장충금/장기수선/장수금`, `아이/어린이/자녀`처럼 동의어와 자주 쓰는 줄임말·오타를 함께 찾을 수 있어야 한다.

## 실제 수정
- `index.html`의 넓은 상황 카드 3개를 구체적인 의도 카드 6개로 분리했다.
- `집` → `누수`, `관리비`, `계약`
- `자동차` → `자동차보험`, `과태료`
- `돈` → `환급`
- 각 카드는 해당 검색어만 `/articles/?q=`로 전달하도록 수정했다.
- 기존 Hero, 검색창, 퀵메뉴, 신뢰영역, Footer는 유지했다.

## 다음 즉시 실행 작업
1. `articles/index.html` 검색 로직을 별도 Brain 검색 엔진으로 분리한다.
2. 제목·요약·카테고리 가중치를 분리하고 본문 전체 단어의 느슨한 포함 검색을 제한한다.
3. 동의어 사전을 추가한다.
   - 장충금, 장수금, 장기수선 → 장기수선충당금
   - 아이, 어린이, 아동, 자녀, 미성년자 → 아이·교육 관련 의도
   - 일배책, 일상배상, 배상책임 → 일상생활배상책임보험
   - 실비, 실손 → 실손의료보험
4. `누수`, `관리비`, `계약`, `자동차보험`, `과태료`, `환급` 검색 결과에 다른 의도의 글이 섞이지 않는지 QA한다.
5. 운영 화면에서 각 카드와 검색 결과를 직접 확인한 뒤 최종 PASS를 기록한다.
