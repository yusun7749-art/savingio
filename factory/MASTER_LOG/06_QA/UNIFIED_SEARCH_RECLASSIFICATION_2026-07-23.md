# Savingio 통합 검색·전체 분류 재구성

날짜: 2026-07-23

## 문제
- `장기수선충당금` 검색이 관리비 상위 그룹의 `전기료·수도료`까지 확장되어 전기요금 글이 노출됨.
- 메인, 정보센터 가운데, 왼쪽 탐색 검색이 같은 결과를 보장하지 못함.
- 글 카드의 기존 대분류가 제목과 실제 검색 의도와 맞지 않는 경우가 있음.

## 실제 수정
- `js/savingio-brain-navigation.js`를 공통 검색 엔진 v19로 교체.
- 상위 분류 단순 확장 방식을 폐기하고 주제별 엄격 검색 의도(INTENTS) 적용.
- 장기수선충당금, 누수, 임대차계약, 자동차보험, 교통사고, 과태료, 아이, 연금, 정부지원, 환급, 세금, 급여, 보험, 은행카드, 전기요금, 통신비, 생활비, 건강, 사업, 부업을 독립 검색 의도로 분리.
- 각 글의 제목·설명·URL·검색 키워드를 기준으로 대분류와 세부 주제를 다시 판정.
- 정보센터 카드, 왼쪽 탐색 결과, 메인 검색 URL이 동일한 점수 함수와 분류 함수를 사용하도록 통합.
- 검색어가 특정 의도로 판정된 경우 다른 대분류 글은 결과에서 제외.

## 기대 결과
- `장충금`, `장수금`, `장기수선` 검색 시 장기수선충당금 관련 글만 표시.
- 전기요금, 누진제, 에어컨 글은 장기수선충당금 결과에서 제외.
- `아이`, `어린이`, `아동`, `자녀`는 동일한 아이·교육 의도로 검색.
- 모든 검색창이 동일한 검색어와 결과 기준을 사용.
