# Savingio Platform DNA V1

최종 갱신: 2026-07-19 KST
상태: LOCK

## 공식 방향

Savingio는 생활정보 블로그가 아니다.
Savingio는 사용자가 실제로 겪는 사건에서 출발해 다음 행동을 안내하는 생활 문제 해결 플랫폼이다.

공식 문장:

> 검색 결과를 보여주는 사이트가 아니라, 생활 문제를 해결하는 플랫폼.

> 사람은 글을 읽으러 오는 것이 아니라, 문제를 해결하러 온다.

## 핵심 헌법

1. 사람은 정보를 찾는 것이 아니라 해결책을 찾는다.
2. 가장 급한 답을 첫 화면에서 제공한다.
3. 전문가 용어가 아니라 사용자가 실제로 검색하는 사건과 문장으로 시작한다.
4. 보험·계산기·정부지원·서비스는 출발점이 아니라 해결 과정의 답이다.
5. 모든 화면은 사용자의 다음 행동을 설계한다.
6. 콘텐츠 단위는 글이 아니라 문제 해결 Flow다.
7. 기본 해결 구조는 5초 결론 → 30초 행동 → 5분 상세 설명이다.
8. 광고는 억지 배치하지 않고 실제 필요가 생기는 해결 단계에서만 연결한다.
9. PC 축소판 모바일이 아니라 Mobile First로 설계한다.
10. 카드 UI를 기본값으로 사용하지 않는다.

## 공식 문제 해결 Flow

사건 발생
→ 5초 결론
→ 지금 해야 할 일
→ 내 상황 선택
→ 맞춤 해결
→ 비용 계산
→ 보험·정부지원 확인
→ 필요한 서비스
→ 관련 사건
→ 심화 정보

## 메인 UX LOCK

- 첫 질문: `오늘 어떤 일이 있으셨나요?`
- 검색창과 상황 선택이 첫 화면의 주인공이다.
- 불필요하게 큰 Hero, 로고 장식, 빈 공간을 만들지 않는다.
- 모바일 Hero 제목은 과장하지 않고 28~31px 수준을 기준으로 한다.
- 모바일 첫 화면에서 검색과 상황 선택 진입이 최대한 빨리 보여야 한다.
- 집·자동차·가족·돈·건강·직장은 카드 묶음이 아니라 서비스 내비게이션으로 표현한다.
- 카드보다 리스트, 단계, Flow, 선형 내비게이션을 우선한다.
- 카드가 필요한 경우는 계산·비교·선택처럼 경계가 실제로 필요한 기능에 한정한다.
- 공간은 장식이 아니라 정보와 행동을 위해 사용한다.
- 배경 기하학은 도트만 남기지 않는다. Savingio S DNA를 반영한 육각형·연결선·노드를 Hero 전체 폭에 깔고, 실제로 보이되 본문을 방해하지 않는 수준을 유지한다.
- 새 디자인을 계속 만들지 않는다. 승인된 메인 구조의 밀도와 완성도를 높인다.

### Hero 공식 카피 헌법 확정

H1:

> 필요한 답을 가장 빠르게.

Description:

> 정부지원, 보험, 생활비, 세금, 환급까지 필요한 정보와 해결 방법을 쉽고 정확하게 안내해드립니다.

LOCK:

- 위 제목과 본문은 Savingio 메인 Hero 공식 카피로 고정한다.
- 디자인·배경·여백·폰트 수정 과정에서 임의로 축약하거나 변경하지 않는다.
- 사용자 명시 승인 없이 다른 문구로 교체하지 않는다.
- 기존 `오늘 어떤 일이 있으셨나요?` 및 임의 축약 문구보다 위 공식 카피를 우선한다.

### 배경 DNA 방향 확정

- Hero 전용 배경으로 제한하지 않는다.
- 배경은 사이트 전체 `body` 영역에 적용한다.
- Hero, 상황 선택, 해결 순서, 서비스, 정보 영역이 하나의 동일한 브랜드 배경 위에서 이어져야 한다.
- 스크롤 이후에도 기하학 DNA가 자연스럽게 연결되어야 한다.
- Hero만 별도로 꾸민 페이지처럼 보이게 만들지 않는다.
- 공식 방향은 `페이지 여러 장을 붙인 화면`이 아니라 `하나의 브랜드 공간 위에 콘텐츠가 놓인 플랫폼`이다.

### 공식 로고·워터마크 사용 규칙

사용자가 이미지로 제시한 기존 Savingio 로고를 그대로 기준으로 한다.

구성:

- 공식 S 심볼
- `Savingio`
- `Smart Life, Better Saving`

적용 원칙:

- 새로운 로고를 임의 제작하지 않는다.
- 공식 로고의 형태와 배치를 유지한다.
- 배경용 적용 시 전체 요소를 같은 블루 계열로 통일한다.
- 기존보다 조금 더 선명하되 콘텐츠보다 먼저 보이지 않는 은은한 수준을 유지한다.
- 로고와 문구는 Hero 오른쪽 상단 브랜드 위치에 맞추되, 배경 자체는 전체 페이지에 적용한다.
- 공식 S 심볼 아래에 `Savingio`와 `Smart Life, Better Saving`을 같은 계열의 은은한 글씨로 배치한다.

### 기하학 배경 구성

- 육각형 네트워크
- 연결선
- 노드
- 일부 다이아몬드 또는 작은 기하학 요소
- 매우 낮은 채도와 투명도
- 콘텐츠 가독성을 해치지 않는 수준
- 사이트 전체를 관통하는 Savingio 브랜드 DNA 역할

### 사용자 피드백에서 확정된 금지 사항

- 사용자가 이미지 제작을 명시하지 않았는데 이미지 생성 도구를 실행하지 않는다.
- 디자인 검토·배치 지시는 이미지 생성 요청으로 해석하지 않는다.
- `실행`, `반영`, `메인에 바꿔놔`는 실제 저장소 코드 수정 지시로 처리한다.
- 사용자가 제공한 공식 로고 이미지를 다른 S 장식이나 새 로고로 대체하지 않는다.
- Hero 카피를 짧게 만들기 위해 임의로 줄이지 않는다.
- MASTER LOG 기존 내용을 덮어쓰지 않는다. 이후 기록은 항상 하단 Append 방식으로 누적한다.
