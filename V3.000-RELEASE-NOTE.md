# Savingio Factory V3.000

## PM One-Click Console

- 실행 파일은 `00-START-FACTORY.bat` 하나로 통일
- 글 생성 시 제목을 직접 입력하지 않고 생성 개수만 입력
- Factory가 자동 주제 풀에서 기존 글과 겹치지 않는 주제를 선택
- 메뉴: 글 자동 생성 / 계산기 점검 / 사이트 점검 / 릴리스 / 전체 자동 실행
- 각 주제 실패를 분리해 나머지 배치가 계속 진행
- 결과 보고서: `factory/output/pm_factory_report.json`
- 작업판: `factory/output/factory_workboard.json`
- Publisher ID는 기존 단일 설정을 그대로 사용하며 변경하지 않음
