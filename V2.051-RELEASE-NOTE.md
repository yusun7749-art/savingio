# Savingio Factory V2.051

## Cloudflare Verify Engine Fix

- Cloudflare API credentials가 없는 Windows 환경에서도 실제 배포 사이트를 자동 확인합니다.
- Cloudflare API timeout/failure 시 `https://savingio.com` 공개 사이트 검증으로 전환합니다.
- `/`, `/articles/`, `/robots.txt`, `/sitemap.xml`이 모두 정상 응답하면 PASS 처리합니다.
- 일시적인 전파 지연을 고려해 기본 6회, 10초 간격으로 재시도합니다.
- API 실패 원인과 공개 사이트 확인 결과를 `factory/output/cloudflare_verify_report.json`에 함께 기록합니다.
- 공개 사이트까지 실패하면 기존처럼 FAIL로 중단합니다.

공식 Publisher ID는 변경하지 않았습니다.
