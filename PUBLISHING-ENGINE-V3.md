# Savingio Publishing Engine V3

입력: 주제, 검색 의도, 대분류, 중분류, 소분류, 공식 근거, 연결 계산기

처리:
1. 7개 글 유형 중 하나를 선택한다.
2. Master Article DNA V3 공통 구조를 적용한다.
3. 유형별 필수 블록을 추가한다.
4. 약 5,000자의 주제 고유 콘텐츠를 작성한다.
5. Explorer 위치와 다음 행동 링크를 지정한다.
6. QA 90점 이상만 발행 후보로 인정한다.

적용 단위:
- 먼저 대분류별 대표 글 1개
- 대표 글 승인 후 해당 대분류 전체
- 전체 일괄 변환 전 링크·레이아웃·모바일 QA 필수
